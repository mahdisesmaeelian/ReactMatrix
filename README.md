# پروژه React ماتریس

یک پروژه React برای **ورود و مرتب‌سازی ماتریس** با قابلیت‌های زیر:

- تعیین ابعاد ماتریس (تعداد سطر و ستون)
- وارد کردن مقادیر ماتریس با اعتبارسنجی عددی
- مرتب‌سازی و نمایش نتیجه
- طراحی ریسپانسیو و زیبا با Bootstrap
